Руководство администратора

В главном меню представлен ряд возможностей представленных на картинке

Кнопка "Начать создание нового сканворда", при нажатии на которую открывается страница ввода параметров для создания сканворда.
Для создания нового сканворда необходимо ввести:
  • Ввести Название сканворда (не менее 4 символов)
  • Ввести ширину сканворда
  • Ввести длину сканворда
  • Ввести количество подсказок для сканворда
  • Выбрать словарь понятий для сканворда
После ввода параметров необходимо нажать кнопку "Создать", после чего начнётся создание сканворда
При нажатии и удерживании левой кнопки мышки (далее ЛКМ) происходит выделение синим цветом, изображающее выбор местоположения ответа на вопрос в сканворде.
После выбора местоположения необходимо отпустить ЛКМ для дальнейшего выбора местоположения вопроса. Вопрос может располагаться только рядом с начальной клеткой ответа.
Для выбора места расположения вопроса необходимо нажать ЛКМ в подсвечиваемых клетках. После чего появиться Список слов, которые можно поставить в клетки.
Если при выборе местоположения ответа или вопроса вы произвели ошибку, то необходимо нажать клавишу "Ecs" на клавиатуре, что приведёт к сбросу выделенных ячеек.
При нажатии на слово "Слово" в списке будет произведена сортировка по Слову от А до Я, при повторном нажатии на слово будет произведена сортировка от Я до А.
При нажатии на слово "Тип" в списке будет произведена сортировка по типу, при повторном нажатии на слово будет произведена обратная сортировка.
При нажатии на тип интересующего вас вопроса будет отображёно содержимое вопроса.
При нажатии на слово интересующего вас вопроса будет произведена подставка слова в выделенные ячейки.
Если при вводе слова нет нужного для вставки, можно добавить слово нажатием кнопки "Добавить", что разрешит добавить слово в словарь, не переходя в раздел работы со словарём.
При вводе ответа разрешен ввод только слов кириллицы и без использования цифр. При вводе вопроса разрешён ввод цифр. При неверном вводе в нижнем правом углу будет отбражён символ.
При верном вводе в нижнем правом углу будет отбражён символ, ввод слова закроется и добавленное слово отобразиться в списке.
Для удаления пар слов и вопросов необходимо включить режим удаления кнопкой "Удалить слово".
После чего нажатием ЛКМ по слову или вопросу произойдёт их удаление. Для отключения режима удаления необходимо повторно нажать кнопку "Удалить слово"
Для сохранения сканворда необходимо нажать клавишу "Сохранить" и в зависимости от завершённости сканворда можно получить ошибку (при попытке сохранить пустой сканворд) или получить сообщение об успешном сохранении сканворда, где при полном его завершении, он будет отображаться пользователям или, если сканворд завершён не полностью, можно будет вернуться к его созданию по нажатию кнопки "Продолжить создание сканворда" в главном меню.

Руководство по авторизации в системе

Для получения доступа к системе необходимо нажать кнопку "Войти", если у вас уже есть аккаунт. Если же вы первый раз в системе, то необходимо выбрать кнопку "Регистрация" для создания нового аккаунта.

При нажатии кнопки "Войти" в поля Логин и Пароль необходимо ввести Логин и Пароль соответственно от аккаунта.
Если же у вас нет аккаунта, вы можете перейти на регистрацию нажатием на текст "Зарегистрироваться".

При нажатии кнопки "Регистрация" необходимо ввести Имя, Пароль и ввести пароль повторно.
При вводе Имени и Пароля необходимо вводить более 4 символов.
При повторном вводе Пароля необходимо ввести те же символы, которые были введены в поле пароля.

После чего необходимо ввести логин и пароль снова, для успешной авторизации.
